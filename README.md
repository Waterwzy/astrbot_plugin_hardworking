## 指令

### 前置知识
本插件使用ISO8610表示专注持续时间：

`ISO8601` 是一种规范的表示时间的格式。本插件使用 ISO8601 表示封禁持续时长。

一个ISO8601持续时间字符串可以是以下格式：

`P1Y2M3DT4H5M6S` 其中：

- `P` 代表周期，**必填。**
  
- `1Y` 为一年。
  
- `2M` 为两个月。
  
- `3D` 指三天。
  
- `T` 用于分隔天和小时，**非必填**。
  
- `4H` 指四个小时。
  
- `5M` 指五分钟。
  
- `6S` 指六秒。
  

上述字符串表示的持续时间为“一年两个月三天四小时五分钟六秒”，再举几个例子：

- `P1M` 指一个月。
  
- `PT2M` 指两分钟。
  
- `P1YT1H` 指一年一个小时。

### 指令详解
本插件所有指令都注册为 `hd`(hardworking)指令组。
1. `/hd set times`用于设定一定时间的专注时间。

- `times`：专注时间持续时长（ISO8601）

2. `/hd fset times`用于设定一定时间（不可取消）的专注时间。

- `times`：专注时间持续时长（ISO8601）

3. `/hd clear`用于取消非强制的专注时间。
